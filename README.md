# Quaddle - Book Your Next Experience

## INSTRUCTIONS TO RUN

1. Clone repository onto your system
```bash
git clone https://github.com/jsaunz/quaddle-SH26.git
```

2. Change directory to the project
```bash
cd quaddle-SH26
```

3. Open the application by running a local web server

4. Or simply double-click `index.html` to open directly in your browser.
